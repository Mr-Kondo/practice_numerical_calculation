## 1. 中央穿孔平板問題

### 1.1 概要
中央に円孔を有する平板を、一軸方向に引張る問題である。
材料力学、弾性力学、有限要素法（FEM）の初歩的かつ重要なベンチマークとして広く用いられる。

この問題の要点は、**孔の周辺で応力集中が発生する**ことである。
均一応力場を与えても、幾何学的不連続である穴の近傍では応力が局所的に増大するため、
数値計算では以下の確認に適している。

- 応力集中の再現性
- メッシュ分割の妥当性
- 要素形状や要素次数の影響
- 境界条件実装の妥当性
- 理論解との比較精度

### 1.2 典型設定
代表的には、以下のような条件で扱う。

- 平板形状: 長方形平板
- 穿孔: 平板中央に円孔
- 荷重条件: 左右端または上下端に一様引張応力または強制変位
- 材料: 線形弾性体
- 解析条件: 平面応力または平面ひずみ

対称性を利用して、全体ではなく 1/2 モデルや 1/4 モデルで解析することも多い。

### 1.3 何を検証する問題か
この問題では、特に**応力集中係数**の再現が重要である。
無限板に円孔がある場合、Kirsch の解として知られる理論解が存在し、
一軸引張下では孔縁で最大応力が遠方一様応力の約 3 倍となる。

したがって、この問題は単なる変形計算ではなく、
**局所的な応力ピークをどれだけ適切に捉えられるか**を確認するための問題である。

### 1.4 数値計算上の着眼点
- 孔近傍では応力勾配が急であるため、局所メッシュ細分化が必要になりやすい
- 低次要素では応力の滑らかさや最大値の再現に限界があることがある
- 節点変位は比較的安定でも、応力分布は粗いメッシュで大きく誤差を持ちうる
- 反力や平均ひずみだけ一致しても、孔縁最大応力が不正確な場合がある


### 1.5 主な用途
- FEM 実装の初期検証
- 応力集中の理解
- メッシュ依存性評価
- 平面応力・平面ひずみモデルの比較
- ポスト処理（主応力、ミーゼス応力等）の確認

### 1.6 概念図
以下は、中央穿孔平板問題の典型的な概念図である。

```mermaid
flowchart LR
    A[左端: 引張荷重または変位境界] --> B[長方形平板]
    B --> C((中央円孔))
    C --> D[孔縁で応力集中]
    D --> E[右端: 引張荷重または変位境界]
```

より幾何学的に表すと、以下のようなイメージになる。

```text
      ← 引張                     引張 →
┌──────────────────────────┐
│                          │
│          ◯               │
│       中央円孔            │
│                          │
└──────────────────────────┘
```

孔の上下左右、とくに荷重直交方向の孔縁近傍で応力集中が生じる。

### 1.7 参考文献
- Timoshenko, S. P., and Goodier, J. N., *Theory of Elasticity*, McGraw-Hill.
- Kirsch, G. (1898), "Die Theorie der Elastizität und die Bedürfnisse der Festigkeitslehre", *Zeitschrift des Vereins Deutscher Ingenieure*.
- Zienkiewicz, O. C., Taylor, R. L., and Zhu, J. Z., *The Finite Element Method: Its Basis and Fundamentals*, Elsevier.

## 2. ダムブレイク問題

### 2.1 概要
ダムブレイク問題は、仕切られていた高水位側の水が、仕切りの除去後に低水位側へ急激に流れ出す問題である。
自由表面流れ、浅水方程式、有限体積法（FVM）、Riemann 問題の代表的ベンチマークとして使われる。

流体計算の立場では、単なる「水が流れる問題」ではなく、
**非線形波の伝播、段波、希薄波、不連続解の捕捉、保存則の実装妥当性**を確認するための問題である。

### 2.2 典型設定
最も基本的な 1 次元ダムブレイクでは、初期時刻に以下のような水深分布を与える。

- 左側: 高水深
- 右側: 低水深または乾床
- 初期流速: 0
- 河床: 平坦
- 摩擦: 無視する場合が多い

このとき、仕切り除去後には、
左側へ希薄波、右側へ段波または前線が伝播する。

### 2.3 何を検証する問題か
ダムブレイク問題では、以下のような項目を検証できる。

- 保存則の実装が正しいか
- 数値流束の妥当性
- 衝撃波・段波の捕捉性能
- 数値粘性の強弱
- CFL 条件の取り扱い
- wet/dry 境界処理の安定性
- 解析解または既知解との一致度

特に浅水方程式では、ダムブレイクは**Riemann 問題の代表例**であり、
Godunov 法、Roe 近似 Riemann solver、HLL 系 flux、TVD / MUSCL などの挙動比較に適している。

### 2.4 数値計算上の着眼点
- 保存形で離散化しないと、段波位置や水量収支が崩れやすい
- 1 次精度では数値拡散が強く、波面が鈍る
- 高次化すると非物理振動が出ることがあり、制限関数が重要になる
- 乾床を扱う場合、負の水深や速度発散を避ける工夫が必要になる
- 可視化の都合で 2D 的な occupancy 表示をしても、本質的な検証は 1D の水深・流量分布で行うべきである


### 2.5 主な用途
- 浅水方程式ソルバの初期検証
- FVM の保存性確認
- Riemann solver の比較
- 自由表面流れの基礎学習
- 洪水波・氾濫解析モデルの簡略検証

### 2.6 概念図
以下は、1 次元ダムブレイク問題の典型的な初期状態と、その後の波の伝播イメージである。

#### 初期状態

```text
水深
↑
│   左: 高水深        │右: 低水深または乾床
│   ┌─────────┐      │
│   │         │      │
│   │         │      │
│───┘         └────────────→ x
        仕切り
```

#### 仕切り除去後の典型挙動

```text
水深
↑
│  希薄波
│  ／＼
│ /  ＼_______   段波・前線
│/            \__/
└────────────────────→ x
```

概念的には、左側には希薄波が広がり、右側には段波または前線が進行する。

### 2.7 参考文献
- Toro, E. F., *Shock-Capturing Methods for Free-Surface Shallow Flows*, Wiley.
- LeVeque, R. J., *Finite Volume Methods for Hyperbolic Problems*, Cambridge University Press.
- Stoker, J. J., *Water Waves: The Mathematical Theory with Applications*, Interscience Publishers.
- Chow, V. T., *Open-Channel Hydraulics*, McGraw-Hill.

## 3. 図と参考文献を踏まえた位置づけ

| 項目 | 中央穿孔平板 | ダムブレイク |
|---|---|---|
| 主対象 | 固体力学 | 流体力学 |
| 代表的手法 | FEM | FVM |
| 主な論点 | 応力集中 | 非線形波・保存則 |
| 解の特徴 | 滑らかな変位場 + 局所応力集中 | 不連続を含む自由表面流れ |
| 主な検証対象 | 境界条件、要素、応力評価 | 流束、保存性、衝撃波捕捉 |

この2題は、数値計算の学習において対照的である。
中央穿孔平板は、**滑らかな場をどう高精度に解くか**を学ぶのに向いており、
ダムブレイクは、**不連続や急勾配を含む保存則系をどう安定に解くか**を学ぶのに向いている。

---

## 4. まとめ

中央穿孔平板問題とダムブレイク問題は、それぞれ固体解析と流体解析における代表的ベンチマークである。

- 中央穿孔平板問題は、応力集中と FEM 実装の妥当性確認に適する
- ダムブレイク問題は、浅水方程式・FVM・保存則系ソルバの妥当性確認に適する

現在の実装では、B02（ダムブレイク）の比較対象は `FVM`, `SPH` である。
標準アニメーション `uv run sec01-animate-b02` は side-view timeseries を出力する。

両者を扱うことで、数値計算における
**滑らかな場の精度検証**と**不連続を含む場の安定・保存性検証**の両方を学ぶことができる。
