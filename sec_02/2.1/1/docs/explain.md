# 平面ひずみFEM比較（CST vs Q4）の解説

## 1. このドキュメントの目的

この文書は、`sec_02/2.1/1/fem_plane_strain_compare.py` が何をしているかを説明し、
このシミュレーションから何が分かるかを整理するための解説です。

対象は次の2つです。

- 初学者: 何を比較して、どこを見ると理解が進むか
- 実装者: どの処理で結果が作られ、どの出力を使って検証するか

## 2. 解析の概要

このスクリプトは、同一の平面ひずみ静的問題を2種類の要素で解きます。

- CST: 3節点定ひずみ三角形要素
- Q4: 4節点双一次四角形要素（2x2ガウス積分）

比較の狙いは、離散化の違いが次の量にどう影響するかを見ることです。

- 最大変位
- 最大ミーゼス応力
- 力の釣合誤差
- 計算時間

## 3. 問題設定（何を解いているか）

- 解析種別: 平面ひずみ
- 材料: `E=30000 MPa`, `nu=0.30`, `thickness=1.0 mm`
- 形状: 1/2モデル（対称面を使った半領域）
- 荷重: 上端に一様下向き面力 `top_traction_y = -1.0 N/mm^2`
- 境界条件:
  - 対称面で `ux = 0`
  - 剛体移動抑制のため1点で `uy = 0`

## 4. 計算フロー（スクリプトの中で何をしているか）

### 4.1 メッシュ生成

- 構造格子を作成
- 開口部に相当するセルを除外
- 上端境界（荷重付与用エッジ）を抽出
- 対称境界節点とアンカー節点を決定

### 4.2 要素ごとの剛性組立

- CSTケース: Q4を2三角形に分割して組立
- Q4ケース: 双一次形状関数と2x2ガウス積分で組立
- いずれも平面ひずみ構成則行列を使用

### 4.3 荷重・境界条件・求解

- 上端面力を等価節点力に変換
- 拘束自由度を設定して線形方程式を求解
- 反力と外力の整合から力の釣合誤差を計算

### 4.4 応力評価と可視化

- 要素中心で `sigma_x, sigma_y, tau_xy` を評価
- ミーゼス応力を算出
- 変位図・応力図を半モデルと鏡像全体モデルで保存

## 5. 出力ファイルと読み方

出力先は `sec_02/2.1/1/outputs/` です。

### 5.1 比較サマリ

- `comparison_metrics.csv`
  - `max_disp_mm`: 最大変位
  - `max_von_mises_MPa`: 最大ミーゼス応力
  - `force_balance_error`: 力の釣合誤差
  - `dof`: 自由度数
  - `solve_time_s`: 求解時間

現在の結果（同梱データ）:

- CST: `max_disp_mm=0.5533`, `max_von_mises_MPa=130.46`, `solve_time_s=0.274`
- Q4: `max_disp_mm=0.5620`, `max_von_mises_MPa=85.64`, `solve_time_s=0.364`

### 5.2 詳細データ

- `nodal_displacement_cst.csv`, `nodal_displacement_q4.csv`
  - 節点座標と `ux, uy, |u|`
- `element_stress_cst.csv`, `element_stress_q4.csv`
  - 要素中心座標と応力成分、ミーゼス応力
- `results_cst.npz`, `results_q4.npz`
  - `nodes, elements, displacements, stress, centers, von_mises`

### 5.3 可視化

- ケース別:
  - `displacement_cst.png`, `displacement_q4.png`
  - `von_mises_cst.png`, `von_mises_q4.png`
- 鏡像全体表示:
  - `displacement_cst_full_mirror.png`, `displacement_q4_full_mirror.png`
  - `von_mises_cst_full_mirror.png`, `von_mises_q4_full_mirror.png`
- 比較サマリ図:
  - `comparison_summary.png`

## 6. このシミュレーションで何が分かるか

### 6.1 一致している点

- 最大変位はCSTとQ4で近い値を示す
- 力の釣合誤差が非常に小さく、実装上の整合性は良好

つまり、全体剛性に関連する応答（変位）は、今回の条件では大きく乖離しないことが確認できます。

### 6.2 差が出る点

- 最大ミーゼス応力はCSTとQ4で差が大きい
- 計算時間は今回の設定でQ4がやや長い

つまり、局所応力評価やピーク応力比較では、要素種類の影響が無視できないことが分かります。

### 6.3 実務上の示唆

- 変位評価を主目的にするなら、両者は近い傾向を示し得る
- 応力集中の評価を重視するなら、要素選択とメッシュ依存性の検討が必須
- 応力ピークを結論に使う場合は、追加でメッシュ収束確認を行うべき

## 7. 制約と注意点

- 本モデルは平面ひずみ・線形静解析の単純化モデル
- 1/2対称モデルと開口近傍の離散化に依存する
- 最大応力はメッシュ密度や要素タイプの影響を受けやすい

## 8. 再現方法

リポジトリルートで実行:

```bash
uv run python sec_02/2.1/1/fem_plane_strain_compare.py
```

実行後、`sec_02/2.1/1/outputs/` にCSV/NPZ/PNGが出力されます。

## 9. まとめ

この比較は、「変位の一致」と「応力の差異」を同時に確認できる教材です。
要素タイプの違いは、特に局所応力の評価で結果解釈に影響するため、
目的変数（変位中心か応力中心か）に合わせた要素選択が重要であることが分かります。
